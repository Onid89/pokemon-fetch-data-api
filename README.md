# Pokémon Fetch API

A simple web application that allows users to search for a Pokémon by name and display its image (sprite) using the [PokéAPI](https://pokeapi.co/). Users can enter a Pokémon's name and see the front-facing sprite if it is available.

## Features

- Fetches Pokémon data from the PokéAPI based on user input.
- Displays the front-facing sprite of the Pokémon.
- Provides error handling if the Pokémon is not found.

## Technologies Used

- HTML5, CSS3
- JavaScript (ES6+)
- [PokéAPI](https://pokeapi.co/)

## Getting Started

### Prerequisites

You will need a web browser to view the project. No additional installations are required.

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/Onid89/pokemon-fetch-api.git
   ```

# Usage

- Type the name of a Pokémon in the input field (e.g., "pikachu").
- Click the Pokemon button.
- The sprite of the Pokémon will appear below the button.
- If the entered Pokémon name is invalid, an error will be displayed in the console.
